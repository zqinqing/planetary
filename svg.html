了解 位图 和 矢量图 的区别。

  • 位图：放大会失真图像边缘有锯齿；是由像素点组成；前端的 Canvas 就是位图效果。
  • 矢量图：放大不会失真；使用 XML 描述图形。

SVG 是矢量图的一种格式用 XML 来描述图形。

SVG 可以用 css 设置样式，也可以使用 js 对 SVG 进行操作。

SVG 的几种使用方法：

  • 在浏览器直接打开
  • 内嵌到 HTML 中（推荐⭐⭐⭐）
  • CSS 背景图（推荐⭐）
  • 使用 img 标签引入（推荐⭐）
  • 使用 iframe 标签引入（不推荐❌）
  • 使用 embed 标签引入（不推荐❌）
  • 使用 object 标签引入（不推荐❌）
